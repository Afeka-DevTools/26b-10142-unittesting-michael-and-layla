**Gemini:**
בטח, הנה טבלה עם פונקציות ה-Assert המרכזיות של JUnit:

| פונקציה | הסבר קצר |
| :--- | :--- |
| `assertEquals` | מוודאת שהערך שקיבלת שווה לערך שציפית לו. |
| `assertTrue` | מוודאת שתנאי מסוים מחזיר אמת (True). |
| `assertNull` | מוודאת שהאובייקט המוחזר הוא ריק (Null). |
| `assertThrows` | מוודאת שהפונקציה זורקת שגיאה (Exception) ספציפית במקרה של קלט שגוי. |
